# 🚀 FluxPro — Integrated Web Management System

**FluxPro** is a web system developed with the goal of centralizing and improving the management of services, products, and registrations in a single modern, responsive, and intuitive environment.

The project was developed within the scope of the **Advanced Web and Mobile Programming Techniques** course at Fatec Americana.

# 📌 About the Project

FluxPro emerged with the proposal to replace manual processes with an efficient digital solution, providing more organization, practicality, and agility in information management.

The platform was designed to offer a simple and accessible experience on different devices, such as desktops, tablets, and smartphones.

# ✨ Features

- Integrated management of people, products, and services
- Registration, editing, and listing of information
- Modern and responsive interface
- Automatic address completion via ZIP Code API
- Display of weather information via API
- Optimized navigation for different screen sizes

# 🛠️ Technologies Used

## 🔹 Back-end
- PHP

## 🔹 Front-end
- HTML5
- CSS3
- Initialization

## 🔹 External APIs
- ZIP Code API
- Weather API

# 🎯 Objective

The main objective of FluxPro is to facilitate the management routine through a modern, organized, and accessible application, reducing the need for manual controls and increasing productivity.
